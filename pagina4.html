Descripción del Proyecto

Este proyecto tiene como objetivo principal desarrollar una aplicación web que permita a los usuarios gestionar sus tareas diarias de manera eficiente. La aplicación ofrecerá funcionalidades como la creación, edición y eliminación de tareas, así como la posibilidad de establecer recordatorios y prioridades.

Tecnologías Usadas

  • HTML5
  • CSS3
  • JavaScript
  • React
  • Node.js
  • MongoDB
